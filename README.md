Este é um projeto simples que eu fiz para estudar a criação de sites. Ele é uma página de lista de tarefas desenvolvido para demonstrar 
a criação e a alternância dinâmica entre temas Claro (Light) e Escuro (Dark) utilizando apenas tecnologias web nativas.

Funcionalidades
Visualização Simples: Exibe uma lista de tarefas pendentes.
Alternador de Temas (Dark/Light Mode): Um botão interativo que altera instantaneamente o esquema de cores da página.
Tema Claro: Fundo verde com texto preto.
Tema Escuro: Fundo preto com texto verde.
Botão Dinâmico: O texto do botão se adapta automaticamente ("Light" ou "Dark") dependendo do tema ativo no momento.

Tecnologias Utilizadas
HTML5: Estruturação da página e dos elementos da lista.
CSS3: Estilização da interface. Destaque para o uso de Variáveis CSS (Custom Properties) no seletor 
:root para gerenciar as cores dos temas de forma limpa e eficiente.
JavaScript (Vanilla): Manipulação do DOM (Document Object Model) para adicionar eventos de clique, 
alternar classes CSS (classList.toggle) e atualizar o conteúdo em texto do botão.
